# How to contribute  
0）如果是新的contributor，可以邮件联系我或者在Issues留言，我要请你；  
1）复制上述的模板，'#'后的数字表示第几个添加的课程信息；  
2）新建一个.md文件，文件的命名为课程的名字，尽量与培养方案一致哦😁；  
3）将1）复制的模板粘贴，填补上各项信息即可；  
4）如果有想要评论的课程可以在某门课的评论区域下添加自己的评论信息；
